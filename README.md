# SPHTechApp
SPHTech Mobile Applications Assignment

## Requirements: 
* Xcode 11.5
* Cocoapods

## Installation
```
$ sudo gem install cocoapods
$ pod install
```

## Architecture:
* Clean Architecture
* MVVM
* Data Binding
* Dependency Injection
* Archive Cache

## Features:
* Display a list of data
* Display a clickable image
* Data Caching

## Includes:
* Unit Tests Coverage 95%
